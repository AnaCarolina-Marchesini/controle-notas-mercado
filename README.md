# 🛒 Controle de Mercado Inteligente

Um gerenciador de despesas de mercado robusto, focado em usabilidade móvel, persistência de dados e controle de metas financeiras.

![Mobile First](https://img.shields.io/badge/Design-Mobile%20First-green)
![Status](https://img.shields.io/badge/Status-Conclu%C3%ADdo-brightgreen)

## 🚀 Novas Funcionalidades

- **Gestão de Meta Mensal:** Defina um orçamento e acompanhe o progresso através de uma barra dinâmica que muda de cor (Verde, Amarelo, Vermelho) conforme o gasto.
- **Exportação de Dados:** Botão dedicado para baixar um resumo completo em `.txt` com todas as notas e o total acumulado.
- **Layout Inteligente:** Campos de data e valor alinhados horizontalmente com botões de ação integrados para máxima eficiência.
- **Persistência Total:** Notas e metas são salvas no `localStorage` do navegador, garantindo que nada se perca ao fechar a aba.
- **Acessibilidade & Semântica:** Código estruturado com tags HTML5 puras (`main`, `section`, `header`, `footer`) e atributos ARIA para leitores de tela.
- **UX de Teclado:** Adição de itens via tecla `Enter` e foco automático após inserção.

## 🛠️ Stack Tecnológica

* **HTML5:** Estrutura 100% semântica.
* **CSS3:** Flexbox, Grid e Media Queries para responsividade extrema.
* **JavaScript:** Manipulação de DOM, cálculos de porcentagem e API de Blobs para download.
* **Google Fonts/SVG:** Ícones escaláveis para interface limpa.

## 📱 Interface Responsiva

O projeto foi construído sob a filosofia **Mobile First**.
- Em **celulares**: Os campos se adaptam para facilitar o toque e a leitura.
- Em **desktops**: O layout utiliza o espaço horizontal para manter os campos e o resumo do total organizados lado a lado.
