# 🛒 Controle de Mercado

Um aplicativo web simples e funcional para gerenciar gastos de mercado, permitindo o registro de notas fiscais com data e valor, cálculo automático de total e persistência de dados.

## 🚀 Funcionalidades

- **Registro de Notas:** Adicione a data e o valor de cada compra.
- **Formatação Automática:** Datas exibidas no formato brasileiro (`dd/mm/aa`).
- **Cálculo em Tempo Real:** O valor total é atualizado instantaneamente ao adicionar ou remover itens.
- **Persistência de Dados:** Utiliza `localStorage` para manter seus dados salvos mesmo após fechar o navegador ou atualizar a página.
- **Interface Responsiva:** Desenvolvido com a metodologia *Mobile First*, garantindo uma ótima experiência em celulares e desktops.
- **Atalhos de Teclado:** Adicione itens rapidamente pressionando a tecla `Enter`.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica.
* **CSS3:** Layout responsivo com Flexbox e Media Queries.
* **JavaScript (Vanilla):** Lógica de manipulação de DOM e cálculos.
* **LocalStorage:** Armazenamento local de dados no navegador.
